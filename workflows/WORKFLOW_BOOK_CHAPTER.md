# 책 챕터 집필 워크플로우

## 개요
제텔카스텐 노트 또는 원고 자료 → 단행본 수준 챕터 집필

## 입력
- `chapter_spec`: 장 번호, 제목, 범위
- `source_notes`: 참조할 제텔카스텐 노트 ID 범위 또는 파일 목록
- `output_file`: 출력 파일 경로
- `style_guide`: 서술 스타일 가이드 (있으면)
- `existing_chapters`: 이미 완성된 챕터 파일 (톤 참조용)

## 실행 단계

### 1단계: 자료 읽기
- `source_notes` 전체 읽기
- 기존 챕터가 있으면 톤/분량 참조

### 2단계: 구조 설계
- 장 내부 소절 구성
- 핵심 서사/논점 결정

### 3단계: 집필
- 서술체: `~다/~했다` (해체)
- 키워드 블록/요약 삭제
- In Medias Res 도입 가능
- 유시민·주경철 내러티브 논픽션 톤
- 분량: 특별 지시 없으면 30~50매 (A4 기준)

### 4단계: 자체 검토
- 톤 일관성 확인
- 기존 챕터와 중복 확인
- 사실관계 원문 대조

## 제약사항
- ❌ 현대 교훈/도덕적 교훈 삽입 금지 (재난정치학의 경우)
- ❌ 체크리스트 형식 금지 (민간위임위탁의 경우)
- ✅ 독자는 해당 분야 실무자 또는 교양 독자
- ✅ 구체적 사례·수치·인용 필수
- ✅ 각주 최소화 (본문에 녹이기)

## 완료 조건
- `output_file`에 완성 원고 저장
- 분량(매수) 보고
- 주요 소절 목록 보고
