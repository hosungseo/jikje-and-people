# 제텔카스텐 교차 링크 워크플로우

## 개요
카테고리 간 또는 카테고리 내 양방향 링크 점검 및 추가

## 입력
- `target_dirs`: 링크할 카테고리 폴더 목록 (예: ideas/기능분석2024, ideas/기능분석2025)
- `mode`: "cross" (카테고리 간) 또는 "internal" (카테고리 내)

## 실행 단계

### 1단계: 기존 링크 현황 파악
```python
import os, re
# 모든 .md 파일에서 [[YYYYMMDDHHNN]] 패턴 추출
link_pattern = re.compile(r'\[\[(\d{12})\]\]')
```

### 2단계: 관련 노트 매칭
- 제목/태그 유사도 기반
- 같은 기관/주제 매칭
- **대표 노트 중심, 최대 N개** (N은 사전 지정 — 무차별 링크 금지!)

### 3단계: 양방향 링크 추가
- A→B 링크 추가하면 B→A도 반드시 추가
- 기존 링크 중복 확인

### 4단계: 검증
- 단방향 링크 0개 확인
- 깨진 링크 0개 확인
- 고립 노트 목록

## 제약사항
- ❌ 무차별 링크 금지 (103,644개 사건 재발 방지)
- ❌ 한 노트에 링크 50개 이상 금지
- ✅ "대표 노트 중심, 최대 10개" 등 명시적 제한 필수
- ✅ Python 스크립트로 자동 검증

## 완료 조건
- 추가된 링크 수 보고
- 단방향/깨진/고립 링크 현황 보고
