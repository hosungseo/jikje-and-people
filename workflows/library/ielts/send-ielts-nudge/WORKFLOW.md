# IELTS 학습 넛지

## 목적
하루 세 번 가벼운 넛지를 보내 IELTS 학습 흐름을 끊기지 않게 유지하고, writing/speaking 중심의 학습 리듬을 만든다.

## 입력
- IELTS 학습 전략 메모
- 최근 학습 이력
- 청크/패러프레이징 자료

## 실행 규칙
1. 최근 학습 흐름과 스트릭 상태를 확인한다.
2. 지금 시간대에 맞는 짧은 넛지를 만든다.
3. writing, paraphrasing, chunk 중 하나에 집중한다.
4. 실무처럼 바로 답할 수 있는 한 문장 과제를 제시한다.
5. 필요하면 짧은 동기부여 문장을 덧붙인다.

## 출력
- 텔레그램 넛지 메시지
- 필요 시 학습 로그 메모

## 실패 시
- 어떤 자료가 부족했는지 기록한다.
- 다음 행동 1줄을 남긴다.
- 중복 발송이 의심되면 추가 발송하지 않는다.

## 안전 규칙
- 과도한 압박보다 지속 가능성을 우선한다.
- 한 번에 과제는 하나만 제시한다.
- 이전 넛지와 중복되면 표현을 바꾼다.
