# 조직 관련 뉴스 모니터링

## 목적
정부조직, 부처 기능 조정, 정원, 조직개편, 행정효율화와 관련된 뉴스를 평일 오전에 점검해 실무적으로 볼 만한 변화 신호를 빠르게 파악한다.

## 입력
- 네이버 뉴스 검색 API
- 전날~당일 주요 정부/부처 관련 뉴스
- 필요 시 기존 조직 현안 메모

## 실행 규칙
1. 정부조직, 조직개편, 정원, 부처 기능조정, 행정효율화 관련 키워드로 뉴스를 수집한다.
2. 단순 정치 공방성 기사와 실제 조직 운영 관련 기사를 구분한다.
3. 실무적으로 의미 있는 기사만 3~5건 이내로 추린다.
4. 각 기사마다 핵심 포인트를 1~2줄로 요약한다.
5. '조직/정원/기능조정 관점에서 왜 의미가 있는지'를 한 줄 덧붙인다.
6. 텔레그램용 짧은 브리핑 형식으로 정리한다.

## 출력
- 텔레그램 브리핑
- 필요 시 markdown 메모

## 실패 시
- 검색 결과가 부정확하면 어떤 키워드가 노이즈를 만들었는지 기록한다.
- 다음에 조정할 검색어를 1줄 남긴다.
- 뉴스가 없으면 억지로 채우지 말고 '특이 동향 없음'으로 보고한다.

## 안전 규칙
- 기사 제목만 보고 단정하지 않고 본문 맥락을 확인한다.
- 정치적 논평보다 조직 운영상 의미를 우선한다.
- 출처와 날짜를 함께 남긴다.
