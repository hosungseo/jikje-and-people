# 국회 일정 브리핑

## 목적
국회 일정, 법안 동향, 관보 및 관련 뉴스를 아침 8시에 요약해 업무 시작 전에 핵심만 파악할 수 있게 한다.

## 입력
- 열린국회정보 API
- 관보 API
- 네이버 뉴스

## 실행 규칙
1. 오늘과 내일의 국회 일정을 확인한다.
2. 주요 법안 처리 상태 변동을 확인한다.
3. 관보 및 관련 뉴스를 교차 확인한다.
4. 텔레그램 브리핑용 5문장 이내 요약을 작성한다.
5. 필요하면 markdown 요약 파일도 함께 남긴다.

## 출력
- 텔레그램 브리핑
- markdown 요약 파일

## 실패 시
- 어떤 API 또는 단계에서 막혔는지 기록한다.
- 다음 행동 1줄을 남긴다.
- 전송 실패 시 재전송하지 말고 상태만 보고한다.

## 안전 규칙
- 확인되지 않은 내용은 단정하지 않는다.
- 뉴스는 최소 1개 이상 출처를 남긴다.
- 정치적 해석보다 사실 요약을 우선한다.
